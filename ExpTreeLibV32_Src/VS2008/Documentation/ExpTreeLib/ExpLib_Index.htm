The Library


ExpTreeLib is a Class Library which provides much of the functionality of Windows Explorer. The Library's central Class is the CShItem Class which is an API based version of the .Net FileSystemInfo Class. The rest of the Library consists of a User Control, ExpTree, other infrastructure Classes, and a number of Classes which implement Drag & Drop for various kinds of Controls. It is not necessary to use the ExpTree in order to take advantage of CShItem and the other Classes in the Library. See the Demo Form frmDragToControl for an example of this.

The visible portion of ExpTreeLib is a User Control, ExpTree, which resembles and provides the functionality of the Left Pane of Windows Explorer. Many applications find it useful as a Control which acts as the .Net FolderBrowserDialog Class. As a Control in in Form, it may be used to navigate the File System, Select a Folder and notify the Form of the Folder that was selected. As illustrated (left pane), it supports normal Windows System Menus on a right click.

An application using the ExpTree Control should Handle ExpTree's primary event, the ExpTreeNodeSelected Event. See the Demo Forms for examples of the Event Handler.

The CShItem Class provides information about File System and other items in the Windows Shell Namespace. The information provided is a superset of the information provided by the .Net Classes FileInfo and DirectoryInfo. Additional information provided includes (via the SystemImageListManager Class) the Icons of those items.

The ListView (right pane) of the illustration shows that it is possible to use ExpTreeLib to mimic the Classic Windows Explorer interface.
