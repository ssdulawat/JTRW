Summary

ExpTree_Demo is a project containing several Forms and supporting Classes used to demonstrate the ExpTreeLib Class Library and User Control. The Demo project's Forms illustrate the use of the TreeView-like ExpTree control for displaying Folders in a structured view and the use of a ListView to display the Contents of a single Folder selected via the TreeView.

An additional Form, frmDragToControl is provided which demonstrates using ExpTreeLib to manage a DataGridView Control without the use of ExpTree.
